# Python & DSA Foundations 🚀

Welcome to the **Python & DSA Foundations** crash course! This repository is designed as a highly structured, 3-week precursor to the advanced AI Backend Curriculum.

If you are a beginner, or if you simply need to dust off your core programming skills, this repo will guide you through the essentials of Python, Object-Oriented Programming (OOP), and core Data Structures & Algorithms (DSA).

## 📅 Course Structure (3 Weeks)

- **Week 1: Python Core & OOP Basics**
  - Master Python idioms, control flow, functions, and the fundamentals of Object-Oriented Programming (Classes, inheritance, magic methods).
- **Week 2: Introduction to DSA**
  - Build your algorithmic intuition with Arrays, Strings, Hash Maps, the Two Pointers technique, and Stacks & Queues.
- **Week 3: Intermediate DSA & Capstone**
  - Dive into Recursion, Linked Lists, and Sorting.
  - **Capstone Project:** Build an In-Memory Key-Value Database with transaction support!

## 🌟 Interactive Visualizations
At the end of every week, you'll find an interactive HTML visualizer to help cement your understanding of the concepts learned. Simply open the `.html` file in your browser!

## ⚙️ Setup Instructions

1. **Clone the repository** (if pushed to GitHub) or open the folder locally.
2. **Create a virtual environment** (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. **Install testing requirements:**
   ```bash
   pip install -r requirements.txt
   ```

## 🧪 How to Practice

1. Open the file for the current day (e.g., `week1_python_oop/day1_fundamentals.py`).
2. Read the problem descriptions and implement your logic in the provided functions (where the `TODO` placeholders are).
3. Run the automated test suite for that day to get instant feedback on correctness, time limits, and complexity:
   ```bash
   python -m pytest week1_python_oop/tests/test_day1.py
   ```

Happy coding, and build a rock-solid foundation!
